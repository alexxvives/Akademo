# Skill: Client Onboarding (Moodle → AKADEMO)

Use this when migrating a new client from Moodle to AKADEMO, or onboarding any academy with bulk import.

## Full Guide
The complete step-by-step migration process is at:
**`docs/onboarding/moodle-migration.md`** — read this first.

## Quick Roadmap (13 steps)

| # | Action | Tool |
|---|--------|------|
| 0 | Find Moodle DB prefix | `SHOW TABLES;` in phpMyAdmin |
| 1 | Export 5 CSVs from phpMyAdmin | enrollments, asignaturas, quizzes, questions, files |
| 2 | Create academy in Admin panel | Admin → Academias → Nueva Academia |
| 3 | Generate xlsx + post-import.sql | `node scripts/moodle-to-excel.js` |
| 4 | Fill prices in xlsx | Open `moodle-migration.xlsx` → Asignaturas sheet |
| 5 | Upload via Admin UI | Admin → Academias → Migración CSV |
| 6 | Run post-import.sql | `npx wrangler d1 execute akademo-db --remote --yes --file=post-import.sql` |
| 7 | Upload PDFs to R2 | `node scripts/ftp-to-r2.js` (needs FTP + R2 creds in `scripts/.env`) |
| 8 | Run import-documents.sql | Run AFTER step 5 |
| 9 | Generate quiz SQL | `node scripts/generate-quiz-sql.js` |
| 10 | Split quiz SQL | Separate assignments + questions, filter ghost IDs |
| 11 | Run quiz-assignments.sql | Run before step 12 |
| 12 | Run quiz-questions-filtered.sql | Run after step 11 |
| 13 | Send welcome emails | Admin → [academy] → Enviar emails de bienvenida |

## Key Files for Each Migration

Per-client onboarding data lives in `docs/onboarding/{client}/`:
```
docs/onboarding/{client}/
  asignaturas.csv          # Exported from Moodle
  enrollments.csv          # Exported from Moodle
  quizzes.csv              # Exported from Moodle
  questions.csv            # Exported from Moodle
  files.csv                # Exported from Moodle
  ftp-progress.json        # Auto-generated by ftp-to-r2.js
  import-documents.sql     # Auto-generated
  post-import.sql          # Auto-generated
  quiz-assignments.sql     # Auto-generated
  quiz-questions-filtered.sql  # Auto-generated
```

## Scripts

```
scripts/moodle-to-excel.js      # CSV → xlsx + post-import.sql
scripts/ftp-to-r2.js            # FTP → R2 upload (needs scripts/.env)
scripts/generate-quiz-sql.js    # quizzes.csv + questions.csv → SQL
scripts/fix-mime-types.js       # Fix wrong MIME types after R2 upload
scripts/re-upload-to-r2.js      # Re-upload if R2 objects are missing
```

## Common Issues

| Problem | Fix |
|---------|-----|
| Wrong MIME types in R2 | Run `node scripts/fix-mime-types.js` |
| R2 objects missing but DB is correct | Run `node scripts/re-upload-to-r2.js` |
| Quiz questions missing ("ghost IDs") | Diff Assignment IDs vs Question.assignmentId in D1; regenerate missing quiz SQL with `scripts/generate-quiz-sql.js` |
| FTP credentials | Store in `scripts/.env` (never commit — gitignored) |
| Classes must exist before quiz SQL | Steps 11-12 must run AFTER step 5 |
